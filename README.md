# Inyector de flujo laminar
En este proyecto se diseño e implemento un inyector para control de flujo laminar, para ello, se realizaron analisis teoricos y de simulación con herramientas CAE considerando parametros clave como caudal, velocidad, número de Reynolds, pérdidas, para estimar el comportamiento del fluido para distintas inclinaciones del inyector. Finalmente, implementando un prototipo funcional con materiales economicos.
